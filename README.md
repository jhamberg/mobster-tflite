# mobster-tflite

1) Download the CORe50 dataset
2) Extract in master/public/
3) Generate listfile with `bash generate-listfile`
4) Run server with:
```
IP=<server ip> node master.js
```

5) Start up the Android client(s)
6) Type "ready" to run the job
